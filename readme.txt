Der active value Front-End Developer Test
-----------------------------------------

Hallo!

Um einen ersten Eindruck zu gewinnen, über welche Kenntnisse Sie in den Bereichen HTML und CSS verfügen, haben wir eine kleine Aufgabe vorbereitet.


-----------
Die Aufgabe
-----------

Sie finden in dieser Ordnerstruktur eine Layoutvorlage für die fiktive Website Muttis Rezeptbuch, die mittels HTML5 und CSS umgesetzt werden soll. Eine Dateistruktur mit Verzeichnissen für CSS, JS und Bilder ist bereits im „www“ Ordner vorhanden, ebenso wie eine reine XHTML-Umsetzung der Website. Ihre Aufgabe besteht nun aus drei Teilen: 


------
Teil 1
------

Konvertieren Sie das vorhandene XHTML-Dokument in ein valides HTML5 Dokument und verwenden Sie, dort wo es sinnvoll ist, neue, semantische Tags und Attribute anstatt der veralteten XHTML Tags.


------
Teil 2
------

Setzen Sie auf dieser Basis das Layout mittels der gelieferten Bilder responsive um. Das mobile Design ist Ihnen hierbei freigestellt (s. Teil 3). Die Verwendung von CSS 3 sollte hierbei selbstverständlich sein. LESS oder SASS ist dabei gerne gesehen. Hierzu finden Sie im Ordner „assets“ eine (Beispiel-) SASS Dateistruktur. Zu Optimieren ist die Umsetzung auf aktuelle Browser und auf einen aktuellen IE Ihrer Wahl. Wichtig: Teilen Sie uns bei Abgabe der Umsetzung die IE Version mit.


------
Teil 3
------

Innerhalb der responsiven Umsetzung sollte die obere Navigation auf Mobilen Endgeräten Auf- & Zuklappbar gestaltet sein. Programmieren Sie diese Funktion auf Basis von Javascript in die Seite ein. Im Ordner „JS“ steht Ihnen hierfür  die Datei „functions.js“ bereit. Gerne kann das auch hierfür eingebundene Framework „jQuery“ genutzt werden.


Senden Sie uns bitte anschließend den Link zu Ihrer fertigen Umsetzung der Website oder eine zip-Datei mit allen nötigen Dateien.
